## Wall Thickness Standards

### Normal Wall Thicknesses

```
WHAT TO EXPECT
──────────────

INTERIOR WALLS (non-load-bearing):
├── Standard drywall both sides: 4.5" - 5"
│   (2x4 stud + 2 layers 1/2" drywall)
├── With plaster (older homes): 5" - 6"
└── With soundproofing: 6" - 7"

INTERIOR WALLS (load-bearing):
├── Standard: 6" - 6.5"
│   (2x6 stud + drywall)
└── Older construction: 7" - 8"

EXTERIOR WALLS:
├── Standard 2x4 + insulation + sheathing + siding: 6" - 8"
├── Standard 2x6 construction: 8" - 10"
├── Brick veneer addition: +4"
├── Stone veneer: +4" to +8"
└── Old solid masonry: 12" - 24"

SUSPICIOUS THICKNESSES
──────────────────────
Interior wall > 8":  Possibly contains ductwork, plumbing, OR hidden space
Interior wall > 12": Almost certainly unusual—investigate
Wall doesn't match adjacent wall: Something different inside
```

### Measurement Technique

```
HOW TO MEASURE WALL THICKNESS
─────────────────────────────

AT DOORWAYS:
├── Measure from door frame edge to edge
├── Subtract door frame thickness (typically 3/4" each side)
├── Result = structural wall thickness

AT WINDOWS:
├── Measure window sill depth
├── This often equals wall thickness
└── Check for discrepancies with interior measurement

BETWEEN ROOMS:
├── Measure from wall surface in Room A
├── Add any closet depths
├── Compare to distance in Room B
└── Difference = wall thickness (or void)
```

## Grid Paper Floor Plan

```
INSTRUCTIONS FOR SKETCHING
──────────────────────────

1. Start with exterior walls
   ├── Measure each exterior wall
   ├── Use scale: 1 square = 1 foot (or 0.5 foot for detail)
   └── Note any jogs, bays, or extensions

2. Add major interior walls
   ├── Locate wall starting points from exterior
   ├── Measure wall lengths
   └── Note door locations and swing directions

3. Add features
   ├── Closets (with depths)
   ├── Stairs (note direction)
   ├── Chimneys and fireplaces
   ├── Built-in furniture
   └── Unusual features

4. Mark measurements
   ├── Label each room with dimensions
   ├── Note ceiling heights
   └── Mark wall thicknesses at key points

5. Identify anomalies
   ├── Circle areas that don't add up
   ├── Note areas for acoustic testing
   └── Mark suspected void locations
```

## House History Checklist

```
RESEARCH CHECKLIST
──────────────────

□ Year built: ___________
□ Original owner: ___________
□ Architectural style: ___________
□ Known additions/renovations: ___________

HISTORICAL FEATURES TO LOOK FOR:
────────────────────────────────

Built pre-1870:
□ Possible servant passages
□ Root cellar (may have hidden section)
□ Ice box access
□ Coal storage areas

Built 1870-1920:
□ Servant stairs/passages
□ Speaking tubes
□ Butler's pantry with pass-throughs
□ Laundry chutes

Built 1920-1933 (Prohibition):
□ Hidden bars
□ Basement compartments
□ Trap doors
□ False walls

Built 1940-1965 (Cold War):
□ Fallout shelters
□ Basement reinforced areas
□ Emergency supply storage

Built 1970s-present:
□ Safe rooms
□ Media vault rooms
□ Hidden storage

DEED RESEARCH:
□ County recorder's office visited
□ Original blueprints obtained (Y/N)
□ Building permit history reviewed
□ Previous owners researched
```
