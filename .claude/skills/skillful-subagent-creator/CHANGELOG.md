# Changelog: skillful-subagent-creator

## v1.0.0 (2026-02-05)

### Created
- 6-step subagent creation process with Mermaid flowcharts
- Skill selection framework (3-tier loading: preloaded, catalog, none)
- 4-section prompt template (Identity, Skill Rules, Task Loop, Constraints)
- Input/output contract JSON schemas
- DAG orchestration patterns (single specialist, chain, fan-out/fan-in, loop, human-in-the-loop)
- DAG node definition YAML schema
- Complete worked example (PR Reviewer subagent)
- Testing protocol with 5 test types
- 4 anti-patterns (skill-less, overload, missing contract, scope creep)
